# Gerenciador de Tarefas

Este projeto é uma aplicação de console em Python para gerenciar tarefas diárias.

## Funcionalidades

- Adicionar novas tarefas
- Listar tarefas existentes
- Atualizar o nome de uma tarefa
- Marcar uma tarefa como concluída
- Remover tarefas já concluídas

## Como executar

```bash
cd "1-Gerenciador de Tarefas"
python gerenciador.py
```

## Observação

O programa utiliza uma lista de dicionários para armazenar as tarefas e oferece um menu interativo no terminal.
