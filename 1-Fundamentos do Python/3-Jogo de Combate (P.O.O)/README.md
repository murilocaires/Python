# Jogo de Combate (P.O.O)

Este projeto demonstra conceitos de programação orientada a objetos em Python por meio de uma batalha entre um herói e um inimigo.

## Conceitos aplicados

- Classes e objetos
- Herança
- Encapsulamento
- Métodos para atacar e receber dano

## Funcionalidades

- Criar personagens com nome, vida e nível
- Realizar ataques normais e especiais
- Exibir detalhes dos personagens
- Simular uma batalha até o fim

## Como executar

```bash
cd "3-Jogo de Combate (P.O.O)"
python jogo.py
```

## Observação

O jogo é uma implementação simples, ideal para estudar como organizar um projeto com classes e métodos.
