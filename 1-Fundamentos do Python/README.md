# Fundamentos do Python

Este diretório reúne três pequenos projetos em Python para praticar conceitos básicos de programação, estruturas de dados e programação orientada a objetos.

## Projetos incluídos

- 1-Gerenciador de Tarefas: aplicação de console para criar, listar, editar e concluir tarefas.
- 2-Agenda telefônica: sistema simples para cadastrar, visualizar, editar e favoritar contatos.
- 3-Jogo de Combate (P.O.O): exemplo de batalha utilizando classes, herança e encapsulamento.

## Como estudar os projetos

1. Entre na pasta desejada.
2. Execute o arquivo Python correspondente.
3. Explore o código para entender a lógica implementada.

Exemplo:

```bash
cd "Fundamentos do Python"
cd "1-Gerenciador de Tarefas"
python gerenciador.py
```
