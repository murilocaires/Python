# Agenda Telefônica

Este projeto é uma aplicação simples para cadastrar e organizar contatos em Python.

## Funcionalidades

- Salvar novos contatos
- Visualizar todos os contatos cadastrados
- Editar informações de um contato
- Favoritar ou desfavoritar contatos
- Listar apenas os contatos favoritos

## Como executar

```bash
cd "2-Agenda telefônica"
python agenda.py
```

## Observação

A aplicação funciona por meio de um menu interativo no terminal, utilizando uma lista para armazenar os contatos.
