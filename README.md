# Repositório de estudos em Python

Este repositório reúne pequenos projetos e exercícios para estudar Python desde os conceitos básicos até a criação de aplicações web com Flask.

## Estrutura do repositório

### 1-Fundamentos do Python
Esta pasta contém projetos simples para praticar:
- lógica de programação;
- estruturas de controle;
- listas, dicionários e tuplas;
- programação orientada a objetos.

Projetos incluídos:
- Gerenciador de Tarefas
- Agenda telefônica
- Jogo de Combate (P.O.O)

### 2-Desenvolvimento com Flask
Esta pasta reúne exemplos de desenvolvimento com Flask, incluindo uma API simples para gerenciamento de tarefas.

Projeto incluído:
- CRUD de atividades

## Como utilizar
1. Entre na pasta desejada.
2. Leia o README específico do projeto.
3. Instale as dependências, quando necessário.
4. Execute o arquivo principal com Python.

Exemplo:

```bash
cd "2-Desenvolvimento com Flask/CRUD de atividades"
pip install -r requirements.txt
python app.py
```

## Objetivo
O objetivo deste repositório é servir como material de estudo para consolidar conhecimentos em Python e desenvolvimento web.
